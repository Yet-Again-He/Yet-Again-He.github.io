Subjective Inflection

The Subjective Inflection is used to indicate the subject of a sentence by adding a suffix to the subject noun.  When dealing with a sentence clause, the last word gets the suffix.

It can also be used to represent a “to be” statement when placed on the noun preceding the object of the verb.

 

When used to indicate a subject:

Ar-Pharazôn Kathuphazgânun azaggara Zigûrada.

“Ar-Pharazôn the Conqueror waged war against Sauron.”

Kalin nakha zadanad.

“The woman came to the house.”

 

Indicating a “to be” statement:

Ar-Pharazônun Kathuphazgân.

“Ar-Pharazôn is a Conqueror.”

 

Subjective Inflections

Strong-I

Neuter: the variable noun undergoes a-fortification, so [a → â], [i → ê], [u → ô], etc.

Common: add -an to the end.

Feminine: add -in to the end.

Masculine: add -un to the end.

 

Strong-II

Neuter: the variable noun undergoes a-fortification, so [a → â], [i → ê], [u → ô], etc.

Common/Feminine/Masculine: add -n to the end.

 

Weak-I:

Neuter: add -a to the end.

Common: add -an to the end.

Feminine: add -in to the end.

Masculine: add -un to the end.

 

Weak-II

Neuter: nothing (the variable vowels are already lengthened, so nothing can be done).

Common/Feminine/Masculine: add -n to the end.
